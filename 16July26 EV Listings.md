# Task Name: Basic Electric Vehicle Current Sale Listings

## Summary:

I need to be able to find all current specified or unspecified EVs within a certain radius of a zip code that is for sale from a dealership.  

Input:

* Year  
* Make  
* Model  
* Radius  
* Zip code

Output

* All EV listings

* How the inputs are entered doesn’t matter

* All EVs nationwide might not be possible, or might be prohibitive at this stage
